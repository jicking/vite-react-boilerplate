# Vite React SPA

Simple preconfigured SPA template (WIP)

## Features

- Simple folder structure
- Css Modules
- Multi env support (dev, staging, prod)
- Set recomended vs code extensions
- Set build github action
- React router v6(hash)
- Prettier
- Nano stores
- Vitetest
